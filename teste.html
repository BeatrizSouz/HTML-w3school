Cabeçalhos HTML

This is heading 1

This is heading 2

This is heading 3

This is heading 6- vai de 1 até 6

h1 define o título mais importante e h6 o menos importante. Definir corretamente um título é importante para que os mecanismos de pesuisa usem os títulos par indexar a estrutura e o conteúdo das páginas na web, além disso, os usuários folheiam um página por seus títulos, que definem a estutura do documento.

É como em um trabalho, onde temos os tópicos e sub-tópicos para cada assunto abortado no trabalho, para a introdução e temas mais importantes colocamos uma formatação grande para definir a importancia desse tópico, seguindo pelos tópicos 'menos importante' ou sub-tópicos que pode ser definidos por h2,h3,h4 etc

Para títulos maiores

É possivel utilizar a propriedade do CSS: style font-size: valor em px, para especificar o tamanho do título e fugir do padrão definido pelo HTML.

Parágrafos

Posso usar hr para definir uma quebra de página, é exibido como uma linha horizontal. É uma tag vázia, o que siginifica que não tem final delimitado por '<' '/' '>'

Além disso posso usar o br para quebra uma linha no parágrafo
Essa também é uma tag vazia, visto que não tem tag final
